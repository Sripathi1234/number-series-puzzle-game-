
# Number Series Puzzle Game (Java)

This project is a console-based Java application that helps users practice number series aptitude problems.

## Features
- Addition series
- Multiplication series
- Fibonacci series
- Square series
- Score tracking
- Replay option

## How to Run
1. Compile: javac NumberSeriesGame.java
2. Run: java NumberSeriesGame
